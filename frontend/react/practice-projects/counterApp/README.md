# Counter App

A simple counter application built with React.

## Features

- Increment the counter
- Decrement the counter
- Reset the counter

## Installation

1. Clone the repository
2. Navigate to the project directory
3. Run `npm install`
4. Run `npm start`

## Usage

1. Click the "+" button to increment the counter
2. Click the "-" button to decrement the counter
3. Click the "Reset" button to reset the counter

## Technologies Used

- React
- JavaScript
- HTML
- CSS
